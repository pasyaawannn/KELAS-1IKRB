Website Kelas ini adalah project web sederhana yang dibuat menggunakan HTML dan CSS. Web ini dirancang untuk menampilkan informasi kelas seperti profil, struktur organisasi, daftar siswa, jadwal, dan pengumuman secara rapi dan mudah diakses.

Project ini bertujuan sebagai media digital kelas sekaligus latihan dasar dalam pengembangan front-end, terutama dalam penggunaan layout, styling, dan desain responsif.
